## 🇫🇷 Français

**Crack The Code** est une application web gratuite et hors ligne pour comprendre les dépendances, réduire les risques et suivre son craving.

- 🌍 5 régions (BE, FR, CH, CA, LU) avec urgences, centres, groupes d’entraide  
- 📚 19 fiches substances + dossier cannabinoïdes exhaustif + 28 Research Chemicals  
- 🎮 Exercices cognitifs & jeux anti-craving (Tetris, respiration…)  
- 📊 Questionnaires (AUDIT-C, WHO-5, motivation) et courbes de craving  
- 🔒 100 % local : aucune donnée envoyée, tout reste dans votre navigateur  

**Utilisation :** téléchargez le fichier HTML et ouvrez-le. Aucune installation requise.

## 🇬🇧 English

**Crack The Code** is a free, offline web app to understand addiction, reduce risks, and track cravings.

- 🌍 5 regions (Belgium, France, Switzerland, Canada, Luxembourg) with local emergency numbers & support  
- 📚 19 substance monographs + in-depth cannabinoid dossier + 28 Research Chemicals  
- 🎮 Cognitive exercises & anti-craving games (Tetris, breathing…)  
- 📊 Validated questionnaires (AUDIT-C, WHO-5, motivation) & craving charts  
- 🔒 100 % local: no data leaves your browser, works fully offline  

**How to use:** download the HTML file and open it. No installation needed.
